# 언제갈래? ✈️

> 친구들과 떠나는 설레는 여행! 우리 언제 떠날지 여기에서 정해봐요.

## 📖 서비스 소개

**언제갈래?**는 친구들과 함께하는 그룹 여행 일정을 쉽고 재미있게 조율할 수 있는 웹 애플리케이션입니다. 복잡한 일정 조율 과정을 간단한 캘린더 기반 투표 시스템으로 해결하여, 모든 참여자가 편리하게 가능한 날짜를 선택하고 최적의 여행 일정을 찾을 수 있습니다.

### 🎯 핵심 가치

- **간편함**: 복잡한 일정 조율을 직관적인 캘린더 인터페이스로 해결
- **실시간 동기화**: 모든 참여자의 선택이 실시간으로 반영되어 즉시 확인 가능
- **AI 기반 추천**: Google Gemini AI를 활용한 맞춤형 여행 일정 자동 생성
- **무료 서비스**: 별도 가입 없이 바로 사용 가능한 무료 서비스

---

## ✨ 핵심 기능

### 1. 실시간 일정 조율
- 캘린더 기반 날짜 선택 시스템
- 가능한 날짜(🟢) / 불가능한 날짜(🔴) 간편 표시
- 모든 참여자의 선택이 실시간으로 동기화
- 최적의 날짜 자동 계산 및 추천

### 2. 친구 초대 및 공유
- 고유한 공유 링크 생성
- 링크 하나로 여러 친구 초대 가능
- 닉네임만 입력하면 바로 참여 가능
- 별도 회원가입 불필요

### 3. 스마트 캘린더
- 한국 공휴일 자동 표시 (2025-2026)
- 주말(토요일, 일요일) 색상 구분
- 여행 기간에 맞춘 자동 캘린더 이동
- 직관적인 시각적 피드백

### 4. AI 여행 일정 추천
- Google Gemini AI 기반 여행 일정 자동 생성
- 여행지, 기간에 맞춘 맞춤형 일정 제공
- 날씨 정보, 일자별 주요 일정, 숨은 명소 추천 포함
- 마크다운 형식의 읽기 쉬운 일정 제공

### 5. 여행지 관리
- 여행 목적지 설정 및 수정
- 최적 날짜 복사 기능
- 일정 공유 및 내보내기

### 6. 사용자 경험 최적화
- 반응형 디자인 (모바일/데스크톱 지원)
- 다크 모드 지원
- 실시간 통계 표시 (현재 계획 중인 여행 수)
- 버튼 클릭 분석 (Analytics)

## 🛠 기술 스택

### Frontend
- **React 19.2.1** - 사용자 인터페이스 구축
- **TypeScript 5.8.2** - 타입 안정성 보장
- **Vite 6.2.0** - 빠른 개발 환경 및 빌드 도구
- **Tailwind CSS** - 유틸리티 기반 스타일링
- **Lucide React** - 아이콘 라이브러리

### Backend & Database
- **Supabase** - 실시간 데이터베이스 및 백엔드 서비스
  - PostgreSQL 데이터베이스
  - 실시간 구독 (Realtime Subscriptions)
  - RESTful API

### AI & Analytics
- **Google Gemini AI** - 여행 일정 생성 AI
- **Vercel Analytics** - 사용자 행동 분석

### 배포 & 인프라
- **Vercel** - 프론트엔드 배포 및 호스팅
- **Netlify** - 대체 배포 옵션 지원

### 개발 도구
- **ESBuild** - 번들링 및 최적화
- **Git** - 버전 관리

## 🚀 주요 특징

### 실시간 동기화
Supabase의 Realtime 기능을 활용하여 모든 참여자의 날짜 선택이 즉시 반영됩니다. 별도의 새로고침 없이 실시간으로 일정 조율이 가능합니다.

### 사용자 친화적 UI/UX
- 직관적인 캘린더 인터페이스
- 색상으로 구분되는 날짜 상태 (가능/불가능)
- 반응형 디자인으로 모든 기기에서 최적화된 경험
- 한국어 완전 지원

### 보안 및 안정성
- 입력 검증 및 Sanitization
- 안전한 API 호출
- 환경 변수를 통한 API 키 관리

### SEO 최적화
- 메타 태그 최적화
- 구조화된 데이터 (JSON-LD)
- Open Graph 및 Twitter Card 지원
- Google Search Console 연동
